### 一、下载源码并解压

```
下载并解压以下源码：nginx源码包 openssl源码包 pcre源码包 zlib-1.2.12源码包
```

### 二、配置编译路径并安装

1. 进入nginx解压的根目录配置编译路径

   ```shell
   ./configure --prefix=/home/user/nginx-1.22.0 --with-pcre=/home/user/soft/pcre-8.45 --with-zlib=/home/user/soft/zlib-1.2.12 --with-openssl=/home/user/soft/openssl-1.1.1p
   
   --prefix:要将nginx安装到哪个目录，新建一个目录好不要和源码解压的目录重复了
   --with-pcre：pcre源码包解压后的路径
   --with-zlib：zlib源码包解压后的路径
   --with-openssl：openssl源码包解压后的路径
   ```

2. 执行编译安装

   ```shell
   //还是在nginx的源码根目录
   
   make && make install 
   
   //执行完在--prefix的目录生成nginx的目录和文件就成功了，源码包都可以删除了
   ```
