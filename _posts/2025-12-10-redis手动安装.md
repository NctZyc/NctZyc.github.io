### 一、下载redis源码解压

下载地址：https://redis.io/download/

```shell
tar -zxvf redis-7.0.11.tar.gz
```

### 二、进入解压根目录指定目录编译安装

```shell
//安装编译环境
yum -y install gcc gcc-c++
//编译安装源码到指定路径
make PREFIX=/home/user/redis7 install 
```

### 三、修改默认配置文件

```shell
1.redis配置文件中有一个bind 127.0.0.1限制ip访问注释掉
2.修改端口 port
3.修改redis让其在守护线程运行 daemonize yes
4.修改进程文件名 pidfile
5.修改日志文件绝对路径和名称 logfile
6.修改持久化文件名 dbfilename
7.修改redis工作目录 dir
8.修改redis密码 requirepass WxSqsIV2nKL3
9.修改Redis的最大内存 maxmemory 1GB
10.修改内存达到上限时的策略为删除即将过期的 maxmemory-policy volatile-ttl
```

### 四、后台启动

```shell
/home/user/redis7/bin/redis-server /home/user/redis7/conf/redis_ggly_8778.conf &
```
